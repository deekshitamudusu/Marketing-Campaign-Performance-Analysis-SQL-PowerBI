Marketing Campaign Performance Analysis using SQL Server \& Power BI



Project Overview



This project analyzes marketing campaign performance across multiple digital marketing channels using SQL Server and Power BI. The objective is to evaluate campaign effectiveness, measure return on investment (ROI), identify high-performing campaigns, and provide actionable business insights through an interactive dashboard.



This project aims to:



1. Analyze campaign performance metrics.
2. Measure revenue generated against marketing spend.
3. Evaluate campaign conversion effectiveness.
4. Identify top-performing campaigns and categories.
5. Build an interactive dashboard for business decision-making.



Dataset Information



The dataset contains campaign-level marketing performance data including:



\* Campaign Name

\* Category

\* Campaign ID

\* Date

\* Impressions

\* Clicks

\* Leads

\* Orders

\* Marketing Spend

\* Revenue
